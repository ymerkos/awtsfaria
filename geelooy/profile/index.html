What are Aliases?

Aliases are alternate names or identities that users can create on the platform. Each alias can have its own profile, allowing users to interact with others using different personas.

What are Heichelos?

Heichelos are spaces or communities within the platform where users can post and interact with each other. Each alias can be part of multiple heichelos, contributing to discussions and sharing content within those spaces.
